🏢 ATKS - Advanced Trading Korea Stocks

한국 주식 시장 전용 완전 자동화 데이 트레이딩 시스템

🎯 핵심 운영 방식: 국내 장 중심으로 여러 지표 분석 → 50개 종목 선정 → 장 시작 전 5개 종목 선발 → 현재 보유자산 최적 배분 매입 → 2-3% 수익 시 매도, -1% 손실 시 손절 → 장마감 전 전체 매도 → 피드백 수치 조절 → 미국장 전날 데이터로 투자 성향 조정

🔧 핵심 기능 및 운영 방식

  • 🔗 실시간 API 연동: 한국투자증권 KIS API 완전 통합
  • 🎯 AI 종목 선정: 시가총액 500억+ → 50개 종목 선정 → 5개 종목 선발
  • 💰 수익/손절 임계값: +2~3% 수익 시 매도, -1% 손실 시 손절
  • 🇺🇸 미국장 연계: 전날 데이터로 긍정적(여유 투자) vs 보수적 투자 조절
  • 🔄 장마감 전 전체 매도 + 피드백 기반 수치 조절
  • 🖥️ 듀얼 인터페이스: Streamlit 웹 대시보드 + PyQt5 네이티브 GUI

⚡ 실제 매매 운영 프로세스

장 시작 전: AI 종목 선별
시가총액 500억+ 기업 → 여러 지표 분석 → 50개 종목 선정 → 당일 투자 5개 종목 선발 (1-5위)
장 시작: 자산 배분 매입
선발된 5개 종목을 수익 가능성 순(예상 수익률)에 따라 현재 보유자산 최적 배분하여 매입
장 중: 수익/손절 자동 실행
2-3% 이상 수익 달성 시 즉시 매도 / -1% 이상 손실 시 손절매매 자동 실행
장 마감 + 피드백: 복기 준비
장마감 전 모든 주식 매도 → 당일 성과 피드백 → 수치 조절 → 미국장 데이터 반영
